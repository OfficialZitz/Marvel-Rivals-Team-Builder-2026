## Marvel Rivals Role Guide

**Vanguard (Tank)** — Frontline anchors. Run 1–2 per team.
**Duelist (DPS)** — Damage dealers and assassins. Run 2–3 per team.
**Strategist (Support)** — Healers and enablers. Always run at least 1.

Running zero Strategists is the most common ranked mistake. The team builder flags this automatically.
